# native_runchen

Flutter与Android原生层交互的插件，内容还在持续扩展中.
现有方法 1、音量控制（音量加、音量减、设置音量、获取当前音量、静音、取消静音
        2、Toast 显示方法
        3、串口通讯（串口开启、串口发送、串口读取、串口关闭
        4、i2c通讯接口（i2c写入、i2c读取
